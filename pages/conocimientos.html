Inicio en el mundo del desarrollo web

A mediados del año 2023 comencé en Coderhouse el curso de desarrollo web frontend react, a medida que fui avanzando en el curso fui adquiriendo diversos conocimientos a través de las clases y las actividades prácticas. A continuación algunos de dichos conocimientos...

Desarrollo Web:

Durante el curso de desarrollo web, adquirí sólidos fundamentos comenzando con los conceptos esenciales de HTML y CSS. Exploramos la estructura inicial de los proyectos y aprendimos a aplicar estilos de manera efectiva a nuestros sitios. Utilicé el preprocesador SASS para organizar y mejorar la legibilidad del código mediante particiones y variables. Además, adquirí habilidades clave como el uso de Flexbox para lograr flexibilidad en la disposición de los elementos de la página, y Grids para implementar el modelo de "grilla" de Bootstrap. También me sumergí en prácticas avanzadas de SEO para optimizar la visibilidad del sitio en motores de búsqueda y aprendí las mejores prácticas de HTML. En cuanto a la gestión de versiones, utilicé Git a través de la terminal integrada de Visual Studio y GitHub para subir y almacenar nuestros repositorios en un servidor global. Esta experiencia me proporcionó un conjunto integral de habilidades para construir y mantener sitios web modernos y eficientes.

JavaScript:

Durante el curso de JavaScript, exploramos los fundamentos esenciales del lenguaje, abarcando conceptos como variables, ciclos, objetos, funciones y arrays, así como funciones de orden superior. Abordamos la utilización de eventos para asignar acciones a elementos, como un botón que redirecciona a otra parte del sitio, y empleamos el almacenamiento local (storage) para preservar datos al recargar o cerrar la página. Además, aprendimos sobre el formato JSON para simular una API, almacenando información en objetos y luego invocando esos datos cuando es necesario. Exploramos operadores avanzados y dominamos la manipulación eficiente del DOM para mostrar elementos en nuestro sitio según nuestras necesidades. Integré el uso de algunas librerías populares como "Bootstrap," "Sweet Alert," y "Toastify." También adquirí habilidades en programación asincrónica para ejecutar bloques de código mientras los usuarios navegan por el sitio, y comprendí el manejo de promesas para gestionar la información de manera efectiva y manejar errores cuando las condiciones no se cumplen. Esta experiencia me ha dotado de las habilidades necesarias para programar de manera eficiente en JavaScript, incorporando técnicas avanzadas y bibliotecas para mejorar la funcionalidad y la experiencia del usuario en mis proyectos.

React Js:

Durante el curso de React js, adquirí habilidades esenciales para el desarrollo web. Aprendí a utilizar la extensión JSX para combinar código JavaScript con elementos adicionales, explorando conceptos como transpilación. Implementamos la práctica de componentes, dividiendo el programa en partes distintas para cumplir funciones específicas. Enfrentamos la programación asincrónica para ejecutar bloques de código en segundo plano y empleamos promesas para manejar tanto la información mostrada en el sitio como posibles errores en condiciones no cumplidas. Utilizamos la función de mapeo (map) para recorrer arrays y mostrar elementos HTML dinámicamente. Abordamos el trabajo con APIs, llamándolas desde nuestro sitio y mostrando la información relevante. Incorporé conceptos avanzados como hooks, children, y patrones, así como la implementación de enrutamiento (routing) para gestionar rutas de navegación. Exploramos eventos para asignar acciones a elementos y el uso de context para una distribución eficiente de información entre componentes sin pasarla por cada uno de ellos. Finalmente, profundizamos en técnicas de rendering y empleamos Firebase para crear y gestionar nuestra propia base de datos en Firestore, integrándola hábilmente en nuestro sitio.
